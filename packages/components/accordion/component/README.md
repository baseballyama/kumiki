# @kumiki/component-accordion

Compound Svelte 5 component (Layer 4) for the Accordion pattern. Generic
over the item value type `V`; named exports `Root`, `Item`, `Trigger`,
`Panel`.

See [`docs/components/accordion.md`](../../../../docs/components/accordion.md) and
[APG Accordion pattern](https://www.w3.org/WAI/ARIA/apg/patterns/accordion/).
