# @kumiki/attachment-accordion

Svelte 5 attachments (Layer 3) for the Accordion component. Provides
`createAccordion(options)` returning a controller with `root` / `item` /
`trigger` / `panel` factories plus imperative methods.

See [`docs/components/accordion.md`](../../../../docs/components/accordion.md) and
[APG Accordion pattern](https://www.w3.org/WAI/ARIA/apg/patterns/accordion/).
