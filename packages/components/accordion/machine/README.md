# @kumiki/machine-accordion

Pure-TypeScript finite state machine for the Accordion component, supporting
single (one-open-at-a-time) and multiple (any-subset) expansion modes plus
roving focus across triggers.

See [`docs/components/accordion.md`](../../../../docs/components/accordion.md) and
[APG Accordion pattern](https://www.w3.org/WAI/ARIA/apg/patterns/accordion/).
