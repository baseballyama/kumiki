# @kumiki/component-slider

Compound Svelte 5 component (Layer 4) for the Slider pattern. Named
exports `Root`, `Thumb`. Single linear value with min/max/step,
horizontal or vertical orientation, RTL inversion of horizontal arrows.

See [`docs/components/slider.md`](../../../../docs/components/slider.md) and
[APG Slider pattern](https://www.w3.org/WAI/ARIA/apg/patterns/slider/).
