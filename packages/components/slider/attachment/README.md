# @kumiki/attachment-slider

Svelte 5 attachments (Layer 3) for the Slider component. Provides
`createSlider(options)` returning a controller with `root` / `thumb`
attachment factories plus imperative methods.

See [`docs/components/slider.md`](../../../../docs/components/slider.md) and
[APG Slider pattern](https://www.w3.org/WAI/ARIA/apg/patterns/slider/).
